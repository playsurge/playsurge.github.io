# 🎮 Surge — Play Games, Your Way

Welcome to **Surge**, the ultimate unblocked game platform built for fun, speed, and style.

🌐 **Live Site:** [https://playsurge.github.io](https://playsurge.github.io)

---

## 🚀 What is Surge?

Surge is a sleek, fast, and modern website for playing a growing library of unblocked web games — right in your browser. Whether you're chilling during a break or sneaking in a game during class (we won't tell 😏), Surge is made for gamers like you.

---

## 🕹️ Features

- 🎮 **Dynamic Game Loader** — Games are stored in a JSON file and load automatically into the site
- 🔍 **Real-Time Search** — Find your favorite game instantly
- 💡 **Auto Theme Detection** — Follows your system theme or toggle manually
- 🎨 **Custom Apple-Style Theme Switcher** — With a purple twist 💜
- ⚡ **Responsive & Mobile-Ready** — Play anytime, anywhere
- 🌈 **Smooth UI & Glow Effects** — Looks as good as it plays

---


---

## 🧠 How It Works

- The `games.json` file holds info about each game: name, description, image, and link.
- JavaScript dynamically loads the games into the homepage.
- Light/Dark themes are applied based on system settings or user toggle.
- Apple-style toggle switch built with pure CSS + JS.
